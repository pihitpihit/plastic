Modifications made by the `plastic` project:
  - Subsetted to Basic Latin + Latin-1 supplement + Latin Extended-A + select
    punctuation/arrows/middle-dot, discarding unused glyphs.
  - Added synthesized glyphs for ASCII control characters (U+0001–U+001F,
    U+007F) and Unicode invisible / formatting characters (U+00A0, U+00AD,
    U+034F, U+180E, U+200B–U+200F, U+202A–U+202E, U+2060–U+2064, U+FEFF).
    Each synthesized glyph has advanceWidth = 3 × "0" advance, and its
    outline is composed of the base font's uppercase letters forming a
    3-character mnemonic (e.g., U+001B → "ESC").

Generated by scripts/build-invisibles-font.mjs.
